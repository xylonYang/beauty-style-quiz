# 你是哪种美？

> 每个人都有属于自己的美，只是你可能还没找到那个词来形容它。

7 道直觉选择题，不到 2 分钟，给你一个答案。

## Why

MBTI 测性格，我们测审美。

不是打分，不是排名，不是"你长得像谁"。
是找到属于你的那一种美，然后用一句话说清楚它为什么好看。

## How

选图，凭直觉。

没有对错，没有标准答案。你选的每一张图，都在悄悄描绘你的审美轮廓。

**两个隐藏维度 x 3x3 矩阵 = 9 种结果**

每种结果都有专属名字、代表画作和一段只属于你的话。

## Features

- 全图片交互，不用动脑，动心就行
- 结果页一键生成分享卡片，发圈即用
- 自带二维码，朋友扫一扫就能测
- 移动端优先，手机体验丝滑
